# Hotel-Revenue-Analysis-Dashboard-Tableau-
Hotel Revenue Analysis Dashboard (Tableau)
Overview
This project analyzes hotel booking and revenue data using Tableau to uncover insights related to revenue trends, customer contribution, and geographic performance.

The objective of this project is to demonstrate practical data visualization techniques, segmentation logic, and analytical reporting suitable for business decision-making.

Project Objectives
Analyze monthly revenue trends
Compare Room Revenue vs Total Revenue using dual-axis visualization
Identify Top 5 and Bottom 5 customers using Sets
Segment customers using Groups
Apply conditional formatting using calculated fields
Rank cities/states based on total revenue
Create a geographical revenue visualization


Tools and Techniques Used
Tableau Desktop
Calculated Fields
Sets (Top 5 / Bottom 5 Customers)
Groups (Gender Classification)
Dual Axis Charts
Table Calculations (Ranking)
Geographic Mapping
Interactive Filters

Visualizations Included
Monthly Revenue Analysis
Bar chart showing revenue trends across months.
Revenue Distribution by Gender
Pie chart representing revenue contribution segmented by gender.
Dual Axis Revenue Comparison
Comparison of Room Revenue vs Total Revenue in a single view.
Customer Segmentation
Identification of Top 5 and Bottom 5 customers based on total revenue.
Revenue Threshold Highlighting
Conditional formatting applied to highlight revenue above a defined threshold.
City/State Ranking
Cities ranked based on total revenue contribution.
Geographical Revenue Map
Map visualization displaying revenue distribution across locations.

Key Insights

Revenue demonstrates seasonal variation, with a noticeable peak in December.
A small group of customers contributes a significant share of total revenue.
Revenue distribution varies across geographic regions.
Conditional highlighting improves performance monitoring clarity.
